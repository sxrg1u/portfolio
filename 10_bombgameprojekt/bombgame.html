Bomb Game

STACK
C# Win-Forms .NET

Über das Projekt

Bomb Game ist ein Minesweeper-inspiriertes Spiel, entwickelt mit C# Windows Forms. Ziel ist es, alle sicheren Felder aufzudecken, ohne eine Bombe zu treffen. Jeder sichere Klick erhöht den Score, während eine Bombe sofort das Spiel beendet.

Spielmechaniken

Der Spieler klickt Felder in einem Raster. Sichere Felder werden grün markiert, Bomben lösen eine Explosion mit Bildschirmblinken und Partikeln aus. Ein Timer misst die Spielzeit, und ein Highscore-System speichert das beste Ergebnis lokal. Das Spiel erkennt außerdem einen Sieg, sobald alle sicheren Felder geöffnet wurden.

Was ich gelernt habe

  • Dynamische UI-Erstellung mit WinForms Panels
  • Zufällige Bombenplatzierung mit 2D-Arrays
  • Spiel-Loop Logik mit Timern
  • Grid- und Kollisionslogik
  • Explosionseffekte mit Screen-Flash
  • Partikel-System in WinForms
  • Highscore speichern mit File I/O
