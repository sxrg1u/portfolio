Snake spiel

STACK
C# Win-Forms .NET RawGameController API

Über das Projekt

Dieses Projekt ist eine Umsetzung des klassischen Snake-Spiels in C# mithilfe von Windows Forms. Ziel war es, ein vollständig funktionierendes Spiel zu entwickeln, das sowohl über die Tastatur als auch über einen Game-Controller gesteuert werden kann. Die Spiellogik umfasst die Bewegung der Schlange, das Erkennen von Kollisionen, das Erzeugen von Äpfeln sowie das dynamische Verlängern des Schwanzes. Zusätzlich wurde ein einfaches Game-Over-System implementiert, das einen Neustart ermöglicht.

Was war die Aufgabe?

Die Aufgabe bestand darin, ein eigenes kleines Spiel zu programmieren, das grundlegende Programmierkonzepte wie Logik, Events, Timer und Objektverwaltung nutzt. Außerdem sollte das Projekt sauber strukturiert sein und eine stabile Spielfunktionalität bieten.

Was ich gelernt habe

  • Arbeiten mit C# Windows Forms
  • Umgang mit Timern und Events
  • Objektorientierte Programmierung in einem echten Projekt
  • Kollisionserkennung und Spiellogik
  • Einbinden eines Game-Controllers über RawGameController
