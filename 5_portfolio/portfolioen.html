About the Project

This portfolio website is my personal presence on the web. It was built from scratch using HTML and CSS and showcases my projects, skills, and background. The design was first created in Figma and then implemented pixel-perfectly. The goal was a modern, dark interface that looks professional while reflecting my personal style.

What Was the Task?

To design and develop my own portfolio website that serves as a digital business card. The design was first created as a mockup in Figma and then cleanly converted into HTML and CSS — with clear structure, good spacing, and a consistent visual system.

What I Learned

  • UI/UX design and prototyping in Figma
  • Semantic HTML and modern CSS layout
  • Building a consistent design system with colors and typography
  • Responsive design for desktop and mobile
  • Going from a Figma mockup to finished browser code
